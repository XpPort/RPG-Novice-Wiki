# Мечи

В Minecraft мечи являются одним из основных оружий, которые игроки могут создавать и использовать для сражений с мобами и другими игроками. Мечи представляют собой острые орудия, которые наносят урон при атаке и могут быть улучшены с помощью зачарований для повышения их эффективности.

В игре существует несколько видов мечей, каждый из которых имеет свои уникальные характеристики. Основные виды мечей в Minecraft включают:
